Hide Deprecated Applications=隐藏不推荐使用的应用程序
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=此设置从可用应用程序列表中隐藏了不赞成使用的应用程序，但是不赞成使用的应用程序仍将显示在已经安装和以前安装的应用程序中
Hide Incompatible Applications=隐藏不兼容的应用程序
This setting will hide applications that are listed as being incompatible with your version of Unraid=此设置将列出隐藏的与你的Unraid版本不兼容的应用程序
Enable Reinstall Default=启用默认重新安装
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=此选项启用卡片和弹出窗口上的另一个按钮，该按钮允许您使用默认模板重新安装已安装的应用程序。此选项有时出现错误，通常安装并同时运行应用程序两次时才使用此选项
Enable additional search results from dockerHub=启用从dockerHub搜索的其他结果
This setting enables CA to retrieve additional search results from dockerHub Note that docker must be enabled for this to be operational=通过此设置CA可以从dockerHub搜索其他的结果，请注意必须启用docker才能使其正常运行
Allow CA to send any emergency notifications=允许CA发送任何紧急通知
This setting allows CA to periodically scan your system and alert you to the presence of=通过此设置CA可以定期扫描系统并向您发出警报
